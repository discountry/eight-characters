# eight-characters

[中文文档](./README_zh.md)

## Installation

```bash
yarn add eight-characters
```

## Usage

```typescript
import { EightCharacters } from "./eight-characters";

const myEightCharacters = new EightCharacters(1990, 1, 1, 12, 1);

console.log(myEightCharacters.bazi())
```
